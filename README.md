A personal portfolio website built with HTML, CSS, and JavaScript, deployed via GitHub Pages. This site serves as a living showcase of my work, skills, and creative projects as a Computer Science graduate with a passion for design, digital storytelling, and animation.

## Setup

To run locally:

```bash
git clone https://github.com/MoonlightNode/website-portfolio.git
cd website-portfolio
```

Then open `index.html` in your browser or use the Live Server extension in VS Code.
